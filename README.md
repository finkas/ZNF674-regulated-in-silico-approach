# ZNF674-regulated-in-silico-approach
For our publication, we identified candidate genes, that are likely regulated by the KRAB zinc finger protein ZNF674 using a comprehensive in silico approach. Therefore, we used publicly available ZNF674 ChIP-seq data generated by Imbeault et al. (2027) (https://www.nature.com/articles/nature21683) curated in the ReMap2022 database (https://remap2022.univ-amu.fr/). The obtained ChIP-seq peaeks were intersected with the GeneHancer "double-elite" regulatory elements (Fishilevich et al. (2017), https://academic.oup.com/database/article/doi/10.1093/database/bax028/3737828) obtained from the UCSC Genome Browser (GRCh37/hg19) and overexpressed transcriptome data of the index case. Only genes, that were not overexpressed in matched controls were kept. All steps were performed using the R script "ZNF674_regulation.R".
